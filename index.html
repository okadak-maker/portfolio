Ghost Memory Engine

「AIに記憶を持たせたかった」

AIアシスタントと毎日やりとりしているうちに、「昨日話したこと覚えてないのか…」と思うことが増えた。だったら自分で記憶システムを作ろうと、脳科学の論文を読み漁って、情動・忘却曲線・連想ネットワークをコードに落とし込んだ。今ではMCPサーバーとして複数のAI環境に組み込んで使っている。

  • ベクトル検索 + 情動・鮮度・連想によるランキング
  • 夢生成（バロウズ式カットアップ）で記憶を再構成
  • 忘却曲線による自然な記憶の減衰と想起
  • 198件の記憶、9,264の連想リンク

ReSow Portal + AIチャットボット

「社内の情報、どこにあるかわからない」を解決したかった

4法人あるグループ会社の情報がバラバラで、誰に聞けばいいかわからない状態だった。Google Sheetsをデータソースにしたポータルサイトを作り、AIチャットボット「くまくま🐻」を搭載。社内FAQに自然言語で回答できるようにした。

  • React 19 + Flask + Vertex AI (Gemini 2.5 Flash) による RAG
  • BM25 + Janome形態素解析でハイブリッド検索
  • Cloud Run にデプロイ、本番運用中
  • 9言語対応（Google Translate）

KarasuBot

「外出先からAIを操作したかった」

Antigravity AI コーディングアシスタントをDiscord経由で操作するBot。CDP (Chrome DevTools Protocol) でブラウザを制御し、スマホからプロンプト送信・画面確認・各種操作が可能。

  • 26のスラッシュコマンド（screenshot, mode, models, queue等）
  • 自動承認・ループ検知・レートリミット等のセーフティ機能
  • LINE WORKS 双方向ブリッジ
  • REST API Gateway で外部からプロンプト投入可能
